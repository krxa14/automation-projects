# Automation Learning Projects

## Overview
A 5-week hands-on learning journey building automation projects with Python, APIs, data pipelines, n8n workflows, and AI agents.

## Weekly Projects

**Week 1: Multi-Source API Aggregator**  
Fetch data from multiple public APIs, merge and analyze results, export to CSV.

**Week 2: Data Validation & Transformation Pipeline**  
Clean messy CSV/Excel files, handle missing values and duplicates, generate summary reports.

**Week 3: Automated Sales Data Pipeline (n8n)**  
Scheduled workflow to fetch, transform, and merge data from multiple sources with email/Slack notifications.

**Week 4: AI Report Writer**  
Use LLMs to generate structured summaries and reports from raw text or CSV data.

**Week 5: Multi-Step Assistant Agent**  
AI agent that breaks down user goals into steps, uses APIs and tools, and logs each action.
